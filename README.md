## 🌟 The Problem

Traditional procurement systems are often fragmented, manual, and inefficient.

Organizations struggle with:

* Vendor information scattered across spreadsheets
* Lengthy approval cycles
* Lack of quotation comparison tools
* Manual invoice generation
* Poor procurement visibility
* Inefficient communication between stakeholders

These challenges lead to increased costs, delayed procurement, and reduced operational efficiency.

---

## 💡 My Solution

VendorBridge AI provides a centralized ERP platform that digitizes the entire procurement lifecycle.

From vendor onboarding to invoice generation, every step is managed through structured workflows, automated processes, and intelligent decision support.

### Core Benefits

✅ Centralized Vendor Management

✅ Automated RFQ Processing

✅ Smart Quotation Comparison

✅ Approval Workflow Automation

✅ Purchase Order Generation

✅ Invoice Management & PDF Export

✅ Real-Time Analytics Dashboard

✅ Procurement Activity Tracking

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/vatsalgajera-tech/Hackathon-2026.git

cd Hackathon-2026
```

## Frontend Setup

```bash
cd client

npm install

npm run dev
```

## Backend Setup

```bash
cd server

npm install

npm run dev
```

## Environment Variables

Create `.env`

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
